<h1>CodeMantra</h1>
<p>CodeMantra is a developer-friendly toolkit designed to help students overcome copy-paste restrictions on coding platforms, making learning and practice smoother and more efficient.</p>

<h2>Method 1: Enabling Copy and Paste in CodeTantra</h2>
<p>The Method 1 specify the particular way of enabling copy and paste in the CodeTantra code space. The following steps you need to follow to enable the copy and paste opt : </p>

<ol>
  <li>Step 1 : Install Firefox Browser (if it's already installed then ignore)</li>
  <li>Step 2 : Login and open your code problem</li>
  <li>Step 3 : Right Click on the Problem Statement </li>
</ol>

<ol>
  <li>Step 4 : Click on Inspect Element </li>
  <li>Step 5 : Click on the Element Selector (in the top left corner of the image)</li>
</ol>
  <img src="./Assests/Screenshot 2026-07-05 103244.png" alt="Inspect Method" width="500">
<ol>
    <li>Step 6 : Take Your Mouse on the Code Space Until It's Showing Full Blue Area</li>
</ol>
  <img src="./Assests/Screenshot 2026-07-05 103650.png" alt="Element Selector">

    <li>Step 7 : Click on the Blue Area and After That You've Noticed in the Inspect Window Line of Code have covered with Blue Line</li>
